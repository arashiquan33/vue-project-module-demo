# VUE工程模块化demo

- 核心模块
  - 登陆模块
  - 主页模块
  - ...
- 特性模块
  - 工单模块
  - 仪表盘模块
- 共享模块
  - 共享组件
  - 共享服务

## 模块化宗旨

每个模块封装与自身模块相关的业务组件/服务或其他,对外通过ES6 MODULE暴漏模块可提供的组件/服务等

## 模块化打包

通过配置webpack与vue动态路由，实现模块化打包，将核心模块和共享模块打包成main,将每一个特性模块打包成异步加载的chunk，将第三方的npm package打包成vendors

## 项目运行

```bash
   npm install && npm start
```
